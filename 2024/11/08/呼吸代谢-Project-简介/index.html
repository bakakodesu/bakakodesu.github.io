呼吸代谢 Project 简介

本项目目的是 DIY 呼吸代谢监测系统来监测人体的呼吸代谢，顺带对人体的 VO2max 进行测量。

具体可以分为 2 个板块：

  1. 数据采集；
  2. 代谢计算。

数据采集

传感器

  1. 传感器的响应时间
  2. 气体流量传感器 AFM3000/SFM3000 驱动
  3. CO2 浓度传感器 SEN0220 驱动
  4. O2 浓度传感器 SEN0496 驱动
  5. Arduino 输入输出模拟信号
  6. Arduino 输入输出数字信号
  7. 文丘里喷嘴通过差压测量流量

上位机

  1. Mac 串口查询之 /dev/cu. 与 /dev/tty. 记录
  2. Linux 串口查询
  3. Python 导出传感器数据到 .csv 文件
  4. Python 绘制数据移动窗口
  5. Python 读取串口数据
  6. 树莓派开启远程桌面并修改分辨率
  7. SCP 远程拷贝文件到树莓派
  8. pyqtgraph 无法实时刷新图表
  9. PyQt 组件之间的层级关系
  10. 呼吸代谢监测：代码

代谢计算

  1. Python 以及 pyqt 线程锁简单使用

  2. 呼吸代谢监测：原理

  3. DIY 呼吸代谢监测系统
